# Twitter Bot

Bot program for [@jser_info](https://twitter.com/jser_info "@jser_info").

## Server

Work on https://gomix.com/#!/project/jser-twitter-bot

## Installation

    npm install

## Usage

    npm start

## Commands

### `@jser /(stat|status|ステータス)/`

https://jser.info/status-of-post の結果を返します。
